VERİ TİPLERİ

Number : number veri tipi tamsayı ve ondalık sayıları temsil eder. Toplama, çıkarma, çarpma, bölme vs. gibi bir çok matematiksel işlem bu veri tipinde yapılabilir. Infinity, -Infinity, NaN gibi değerlerde alabilir.
BigInt : Javascriptte Number veri tipinin temsil edebileceği sayılar 253-1 ile -253-1 aralığındadır. Bu aralık dışında olan sayıları temsil etmek için bigint veri tipi kullanılır. const bigInt=123456789123456789123456789123456789n şeklinde tanımlanır. Sayının sonuna eklenen "n" harfi bigint olduğunu gösterir. Bigint ile yapılan işlemler Number veri tipine göre çok daha yavaş olduğundan zorunlu olmadıkça kullanılmamalıdır.
String : Metin değerlerini saklamak için kullanılabilir. " ", ' ' veya ` ` işaretlerinin içerisine yazılan değerler string veri tipindedir.
Boolean : Boolean veri tipi sadece iki değer alır. true, false
null, undefined : Null ve undefined var olmayan bir veri tipini ifade etmek için kullanılır.
Object : Object nesneleri ifade eder. Referans veri tipidir. Yukarıda belirtilen diğer veri tipleri ise primitive(ilkel) veri tipleridir.

MATEMATİKSEL OPERATÖRLER

+,-,*,/ : Sırasıyla toplama, çıkarma, çarpma ve bölme işlemi yapan operatörlerdir. Number veri tipine uygulanırlar. Ayrıca "+" operatörü String veri tipine uygulanırsa iki stringi birleştirip tek string yapar.
%,** : % operatörü mod almak için kullanılır. (5%2); işleminin sonucu "1" dir.

** operatörü ise üslü sayıları ifade eder. 5**2 işleminin sonucu "25" dir.
++,-- : Bu operatörlar sayı değerini bir artırmak/azaltmak için kullanılır. let x=1; x++; işlemi x değerini 2 yapar.

MANTIKSAL OPERATÖRLER

|| (OR) : Java,C# gibi programlama dillerinde iki boolean değer üzerinde işlem yapar. Javascriptte ise bu işleme ek olarak ilk (truthy) doğru değeri bulmak için de kullanılabilir.

Falsy (yanlış) dönen değerler: false,0,"",null,undefined,Nan değerlerdir.. Bunun haricindeki tüm değerler javascriptte truthy olarak değerlendirlir.

Özetle || her iki değer false ise false döner. En az bir tane true ise true döner.
&& (AND) : Her iki değerin true olduğu durumlarda true değeri döner. En az biri false ise false döner. Örnek: true && true = true; false && true=false;

Ayrıca ilk falsy değeri bulmak içinde kullanılabilir.
! (NOT) : Boolean değerleri tersine çevirir. !true=false; !false=true
