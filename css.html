BAZI CSS SEÇİCİLER

.nav-link : nav-linkler sınıfına sahip bütün html elementlerini seçer. Örn: <div class="nav-linkler">
.c1.c2 : c1 ve c2 sınıfına sahip tüm elementleri seçer. Örnek: <div class="c1 c2">
#test : id'si test olan elementi seçer. Örnek:<div id="test">
p,td, vs : Girlen değeri sağlayan tüm elementleri seçer. Örnek: p değeri girilmişse HTML dökümanındaki tüm <p> elementlerini seçer.
p.c1 : c1 sınıfına sahip olan tüm p elementlerini seçer. Örnek:<p class="c1">

BAZI CSS ÖZELLİKLERİ

heigth, max-height : Sırasıyla elementin yüksekliği ve maximum yüksekliğini belirtir. Girilen değer pixel, oran cinsinden olabilir. Örnek: height:300px; max-height:50vh
width, max-width : Sırasıyla elementing genişlik ve maximum genişliğini belirtir. Örnek: width:100px; max-width:90%
padding : Elementin içeriğiyle çevreleyen kutu arasındaki boşluğu belirler. Tek değer girilirse üst,sağ,alt ve sol boşluğu belirler. Örnek: padding:3px 3px 3px 3px;
margin : Elementin komşu elementlere olan boşluğunu belirler. Padding element içerisinde boşluk bırakırken margin element kutusu dışında boşluk bırakır. Kullanım şekli padding gibidir.
text-align : Girilen metnin nasıl hizalanacağını gösterir. center, left, right ve justifiy değerlerini alabilir. Örnek: text-align:justify; metnin her iki yana yaslanmasını sağlar.
border : HTML elemanın sınırının nasıl olacağını belirler. Üç değer alır. border-width:çizgi kalınlığı, border-style:çizgi stili ve border-color çizgi rengini belirler. Örnek: border: 2px solid red;
cursor : Elementin mouse şeklinin nasıl görüneceğini belirler. Alabileceği değerlerden bazıları: pointer, none, help move vb. Örnek: cursor:pointer;
color : Elemanın rengini belirler. Renk değeri tanımlı renklerden biri olabileceği rgb cinsinden de girilebilr. Örnek: color:rgb(200,80,43);
display : Elemanın nasıl görüneceğini belirler. Alacağı değerler: none,inline,block, inline-block. Örnek:display:inline-block;
list-style-type : Listelerdeki stili belirlemek için kullanılır. daire, kare, roma rakamı, sayı vs olabilir. Örnek: list-style-type:upper-roman;
